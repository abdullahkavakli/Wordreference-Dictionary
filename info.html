WordReference Dictionary

Multi-language dictionary powered by WordReference.com.

Select a word and use Alt + double-click, Alt+Q (page shortcut), or Alt+X (Chrome command shortcut).

Quick start

  • Hold Alt and double-click a word to open the translation bubble.
  • Use Alt + Q or Alt + X on selected text for quick lookup.
  • Press Alt + Z to open the floating dictionary.

In-page popup trigger

To open the in-page translation bubble quickly, you can double-click a selected word while holding a modifier key, or use a keyboard shortcut.

  • Alt + X – Chrome command shortcut for the in-page translation bubble, useful when the page shortcut is unavailable
Shortcuts not responding?
Chrome may have assigned one or more shortcuts to another extension, or left the suggested keys inactive. If that happens, open chrome://extensions/shortcuts, find WordReference Dictionary, and assign the popup and in-page bubble shortcuts manually.
If a page shortcut is still blocked, the Chrome command shortcut usually continues to work.

Features

  • Toolbar popup with manual direction toggling for 18 languages.
  • Select a word on any page and use Alt+double-click, Alt+Q, or Alt+X to open the in-page translation bubble.
  • Right-click any selected text → WordReference: … to open the full results page.
  • Native audio buttons and text-to-speech fallbacks for 18+ dialects and accents.
